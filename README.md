# Walkover-Task
 
#### Emulating Tote bet with custom stakes.
 
## Solution
  
+  Setup Golang dev environment.

+  Add GOROOT and GOPATH to path.


#### Execution
```
git clone https://github.com/code4kunal/golang-tote-bet.git
```

```
cd golang-tote-bet
```

```
go run main.go -result=R:2:3:1:4
```
